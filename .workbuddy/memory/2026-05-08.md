### 项目结构
- **前端**：`d:\专高五\全栈开发\frontend`（Next.js 16 + TypeScript + TailwindCSS + axios）
- **后端**：`d:\专高五\全栈开发\backend`（FastAPI + SQLAlchemy + PyJWT + bcrypt）
- **数据库**：MySQL，数据库名 `fullstack_db`（4张表已自动创建：users/categories/admins/products）

### 后端已实现模块
- `app/core/`：config、security（JWT+bcrypt）、exceptions、response（统一格式）
- `app/models/`：User、Category、Admin、Product（SQLAlchemy ORM，含关联关系）
- `app/schemas/`：UserCreate/Update/Response、LoginRequest/TokenResponse、CategoryCreate/Update/Response、ProductCreate/Response
- `app/services/`：UserService、AuthService（含admin自动初始化）、CategoryService、ProductService
- `app/routers/`：user、auth、category、product（全部接入JWT守卫）
- `app/middleware/`：LoggingMiddleware（全局日志中间件）
- `app/dependencies/auth.py`：get_current_admin（JWT守卫依赖注入）
- `app/main.py`：CORS、日志中间件、全局异常处理、路由注册、启动初始化
