# 将项目上传到 47.98.101.195

在 **本地 PowerShell** 执行（先 SSH 连上过一次并接受指纹）。

## 1. 在服务器创建目录

```bash
ssh root@47.98.101.195 "mkdir -p /var/www/fullstack"
```

## 2. 上传项目（排除 venv 和 node_modules）

在 `D:\专高五\全栈开发` 目录执行：

```powershell
scp -r backend frontend deploy root@47.98.101.195:/var/www/fullstack/
```

> 若 `scp` 较慢，也可用 WinSCP / FileZilla 图形界面上传 `backend`、`frontend`、`deploy` 三个文件夹。

上传后服务器路径：

```
/var/www/fullstack/backend
/var/www/fullstack/frontend
/var/www/fullstack/deploy
```

## 3. 不要用本地上传的 venv

服务器上重新创建虚拟环境（见《服务器上线计划.md》阶段五）。
